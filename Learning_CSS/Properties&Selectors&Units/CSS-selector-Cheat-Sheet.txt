Cascading Style Sheets at the most basic level it indicates that the order of CSS rules matter. 


.class
#id

element
element, element
element1 element2      note* select all element2 inside element1
element1 > element2    note* select all element2 that have parent of element1
element1 + element2    note* select any element2 that exatcly after element1
A ~ B                  note* selects all B that follow a A
:hover

:first-child
